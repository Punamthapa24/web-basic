About ASP.NET MVC

ASP.NET MVC is a web application framework developed by microsoft, which follows the model-view-controller(MVC) architectural pattern.

Application into three interconnected components:

  • Model: Represents the application's data and business logic.
  • View: Responsible for displaying the user interface and presenting data from the model.
  • Controller: Manages user input, updates the model, and returns the appropriate view.

Razor View Engine is used because it provide a clean, efficient, and developer-friendly way to create dynamic web pages in ASP.NET MVC framework:
