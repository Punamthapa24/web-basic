To install Git properly, follow these steps .
  1. Download the Git Installer:
    • Go to the official Git website: https://git-scm.com/downloads
    • Click on the “Download” button for Windows.
    • The download should start automatically. If not, select the correct version for your system (32-bit or 64-bit).
  2. Run the Git Installer
    • Once the file is downloaded, double-click to run the .exe installer.
    • You’ll see the Git Setup window appear.
  3. Now check you've Visual studio on your pc or not. If not then click on Download button and download it.
  4. Visit GitHub’s Signup Page.
    • Open your web browser and go to https://github.com/
  5. Open github and check if you've already have any account or not. If you already have an account then you simply go for sign in button. But if you don't have any account then you must have to sign up first.

    • Enter Your Email Address (Provide a valid email address that you can access. GitHub will use this to send account-related notifications).
