  1. What is CSS?

    • CSS stands for Cascading Style Sheets which is like dressing up a webpage. It makes your webpage look nice by adding colors, changing fonts, adjusting sizes, and arranging things neatly. While HTML builds the structure, CSS handles the style. Think of it as the design magic that makes a boring webpage beautiful!

  2. Why CSS?

    • We use CSS to make websites look good and work well. Without CSS, a webpage would just be plain text and images stacked together. CSS brings in the colors, fonts, layouts, and other styles that make a website attractive and easier to use. It's like adding personality and flair to an plain design!

  4. What are CSS syntax?

    5. =>Here's the basic structure of CSS syntax:

                        selector {
                        property: value;
                        property: value;
                      }

    • Selector:

      Specifies which HTML element(s) the style will apply to (e.g., p, h1, .class-name, #id-name).

    • Property:

      Defines the aspect of the element to be styled (e.g., color, font-size, margin).

    • Value:

      Assigns the specific style to the property (e.g., red, 16px, 10px). For example:

    For example:

                        color: blue;
                        font-size: 18px;
                      }
