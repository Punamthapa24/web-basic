Welcome to Day 1

  1. What is HTML?

    • HTML stands for HyperText Markup Language, and it's like the building blocks of the web. In simple terms, it’s the code used to create and structure content on web pages, like text, images, links, and buttons.

  2. Why HTML?

    • HTML is essential because it’s the foundation of every webpage you see online. Without HTML, there would be no structure or way to organize content on a website. It tells the browser how to display things like headings, paragraphs, images, and links. In short, HTML helps turn plain text into a webpage.

  3. Syntax of HTML:

    4. The basic syntax of HTML consists of elements that are written using tags enclosed in angle brackets (< >). Here's the general structure of an HTML document:

    • <!DOCTYPE html>: Declares the document type as HTML.
    • <html>: The root element of the document.
    • <head>: Contains metadata (e.g., title, styles).
    • <title>: Sets the page's title (shown in the browser tab).
    • <body>: Contains the visible content, like headings, paragraphs, images, etc.

  4. Text Formatting Tags:

    • <h1> to <h6>: Headings (H1 is the largest, H6 is the smallest).
    • <p>: Paragraphs.
    • <b>: Bold text.
    • <i>: Italic text.

  5. Table Tags:

    • <table>: To create tables.
    • <tr>: Table rows.
    • <td>: Table data cells.

  6. Commonly Used HTML Attributes:

    • <id> :Gives a unique identifier to an element.
    • <class>: Groups elements for styling or scripting.
    • <style>: Adds inline CSS to style the element.
    • <href>: Specifies the URL for links.
    • <src> : Defines the source file for images, videos, or scripts.
    • <title>: Adds a tooltip that appears on hover.
    • <name>: Identifies form elements for submission.
    • <placeholder>: Adds hint text inside form fields.
  7. HTML image tags:
    • <a href = "URL">: Creates a hyperlink.
    • <img src = "URL" alt = "description"> :Embeds an image.
